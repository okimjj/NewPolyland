# NewPolyland<br/>
<br/>
<br/>
1. 사용언어 : Java<br/>
<br/>
2. 입력 정보:<br/>
ㄴa. 권종 입력(종일 / 야간)<br/>
ㄴb. 주민번호 7자리(생년월일 + 성별코드) 입력<br/>
ㄴc. 원하는 수량<br/>
ㄴd. 우대 할인 대상 확인<br/>
ㄴe. 발권 과정 반복 여부 확인<br/>
<br/>
3. 작동 개요<br/>
권종에 따른 연령대 별 티켓 가격이 설정돼있다. 주민번호를 통해 산출한 나이로 그에 해당되는 티켓 가격을 뽑고 그 가격에 원하는 수량과 우대할인 여부에 따른 계산으로 가격을 산출한다. 발권 과정이 한 사이클 끝나면 다시 반복할 것인지 확인한다.
